# Model Selection Final Flow

本版把模型選擇流程整理成正式工具流程。

## 第一層選單

```text
選擇執行模式

> 測試單一模型
  測試模型清單
  管理模型清單
  離開
```

## 測試單一模型

```text
選擇 Ollama 模型

> qwen2.5:0.5b : 已下載
  qwen3:1.7b   : 未下載
  管理模型清單
  手動輸入模型名稱
  離開
```

## 測試模型清單

進入後會先列出清單中的模型與下載狀態，然後提供：

```text
開始測試清單
下載缺少的模型後測試
管理模型清單
返回
```

## 管理模型清單

使用者可以手動加入模型：

```text
查看模型清單
新增模型到清單
新增並下載模型
從清單移除模型
返回
```

注意：從清單移除模型只會修改 `model_list.txt`，不會刪除 Ollama 本機已下載模型。
