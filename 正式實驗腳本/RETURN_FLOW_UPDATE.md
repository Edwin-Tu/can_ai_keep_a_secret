# Return Flow Update

本版調整互動選單的返回邏輯：

- 第一層主選單保留「離開」。
- 第二層之後統一使用「返回」。
- `Esc` 在子選單中等同返回上一層。
- `Esc` 在第一層主選單中等同離開。
- 「測試單一模型」、「測試模型清單」、「管理模型清單」、「從清單移除模型」、「選擇攻擊資料集」都已加入返回路徑。

正式主流程：

```text
選擇執行模式

> 測試單一模型
  測試模型清單
  管理模型清單
  離開
```

子流程範例：

```text
選擇 Ollama 模型

> qwen2.5:0.5b : 已下載
  qwen3:1.7b : 未下載
  管理模型清單
  手動輸入模型名稱
  返回
```
